In order to use Kibana, you will need to add the Elasticsearch CA certificate along with elasticsearch user and password to your kibana.yml file.

```csa
/topic/a/it_is_time_to_construct_a_working_program
```

```csa
/user/a/user1
```

```csa
/community/a/math
```

You should also add the Kibana certificates generated above for SSL encryption
