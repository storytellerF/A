为了能够清晰地传达数学表达式，GitHub 支持 Markdown 中的 LaTeX 格式数学。有关更多信息，请参阅Wikibooks 中的LaTeX/数学。

GitHub 的数学渲染功能使用 MathJax；一种基于 JavaScript 的开源显示引擎。MathJax 支持各种 LaTeX 宏和多种有用的辅助功能扩展。有关更多信息，请参阅MathJax 文档和MathJax 辅助功能扩展文档。

数学表达式渲染可在 GitHub Issues、GitHub Discussions、拉取请求、wiki 和 Markdown 文件中使用。
